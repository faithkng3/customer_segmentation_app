# Customer segment classification - Random Forest Model

This is a machine learning streamlit web application that uses Random Forest Model classifier to predict the segment that a customer belongs to based on the features:
1. Age
2. Spending Score(1-100)
3. Work Experience(years)
4. Family size
5. Annual Income

# instalation

To run this project, you need to have Python 3.12.0 or higher installed on your machine. Yo can install the required packages by running the following command

Steps
1. create the virtual environment in this case its named as -- .venv using this command:
      python -m venv .venv

2. Activate the virtual environmentusing the command
    .venv\Scripts\activate or cd .venv --cd .venv-- \Scripts\activate
     or
      source .venv/bin/activate
3. Install the required packages using the command:
   pip install -r requirements.txt
